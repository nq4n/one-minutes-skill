# **App Name**: OneMinuteSkill

## Core Features:

- Video Feed: Vertical video feed displaying 1-minute skill videos.
- Category Filter: Category filter (Critical thinking 
Communication 
Problem-solving 
Adaptability 
Time management 
Team work) to refine video discovery.
- Contributor Profile: Profile page for contributors showcasing their skills and videos.
- Save Video: "Like" or "Save for Later" functionality for easy video organization.
- Personalized Recommendations: AI tool to suggest relevant skill videos based on user's saved content and interactions.

## Style Guidelines:

- Primary color: Teal (#008080) to convey a sense of calm and learning.
- Background color: Light gray (#F0F0F0) to create a clean and modern atmosphere.
- Accent color: Mustard yellow (#FFDB58) to highlight important elements and add a touch of playfulness.
- Body and headline font: 'PT Sans', a humanist sans-serif, combining a modern look and some warmth or personality.
- Use playful and intuitive icons for categories, profiles, and interactions.
- Clean and simple layout to prioritize usability and content discovery.
- Subtle animations on video transitions and interactive elements to enhance user experience.
